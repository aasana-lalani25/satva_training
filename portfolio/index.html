FirstTalk Mobile App Project

FirstTalk is a mobile application developed to improve communication accessibility for the Deaf community. The app integrates machine learning and computer vision techniques using TensorFlow, OpenCV, and Keras, along with a Flutter-based frontend and PHP–MySQL backend. Co-developed as part of a 4-member team, focusing on inclusivity and real-world impact.

Smart Irrigation System Project

Developed an IoT-based smart irrigation system for agriculture using soil moisture sensors, a microcontroller, and wireless connectivity modules to automate water supply based on real-time field conditions. The system helps optimize water usage, improve crop efficiency, and reduce manual intervention. This project was developed as part of a team of 8 and presented at the Azadi Ka Amrit Mahotsav Hackathon 2022.
